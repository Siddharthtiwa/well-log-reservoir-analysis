# well-log-reservoir-analysis
Well Log Based Reservoir Characterization using Gamma Ray and Shale Volume Analysis
Project Overview

This project involves petrophysical analysis of well log data to identify subsurface lithology and potential reservoir zones. Gamma Ray (GR) log is used to interpret lithology variation, and Shale Volume (Vsh) is calculated to differentiate reservoir and non-reservoir intervals.

Objectives
Analyze Gamma Ray log with depth
Compute shale volume (Vsh) from GR data
Identify reservoir and shale zones
Visualize subsurface lithological variation
Methodology
Load well log dataset containing GR and depth
Calculate Vsh using linear GR normalization method
Classify zones based on Vsh cutoff (clean sand and shale)
Visualize GR log and reservoir distribution
Results
Identified alternating sandstone and shale layers
Delineated potential reservoir zones using Vsh
Interpreted heterogeneous depositional environment
Established reservoir–seal architecture
Tools Used

Python, Pandas, Matplotlib, Jupyter Notebook

Files
display.ipynb: Main analysis notebook
WellData.csv: Input dataset
